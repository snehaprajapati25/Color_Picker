# 🎨 Color Studio 



A modern and interactive **Color Studio** built using **HTML5, CSS3, and Vanilla JavaScript**. This application allows users to explore colors using a custom rainbow hue slider, view real-time color previews, convert HSL colors into HEX format, and save their favorite colors in a dedicated saved palette section.



---



## ✨ Features



* 🌈 Custom rainbow hue slider for smooth color selection

* 🎨 Real-time color preview display

* 🔄 HSL to HEX color conversion

* 📝 Automatic HEX code generation and display

* 💾 Save favorite colors with their HEX values

* 📂 Separate Saved Colors tab for managing selected colors

* 🖱️ Interactive and responsive user interface

* 🌙 Modern dark-themed UI with smooth animations



---



## 🛠️ Technologies Used



* **HTML5** – Structured and semantic layout

* **CSS3** – Styling, gradients, animations, and responsive design

* **Vanilla JavaScript** – DOM manipulation, event handling, dynamic color updates, and HSL to HEX conversion



---



## 🧠 How It Works



1. The hue slider generates a color value based on the HSL color model.

2. JavaScript captures the selected hue and updates the live color preview.

3. The HSL value is converted into a HEX color format using a custom conversion function.

4. The generated HEX code is displayed to the user.
